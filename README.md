# 🍺 Arsenal AR Tabletop

WebAR гра з фізикою для бару "Арсенал 2.0" у Чорноморську.

## 🎯 Фічі

- **Image Tracking** через MindAR — працює на iOS та Android
- **Фізика** Cannon-es — реалістичні відскоки та імпульси
- **2 режими**: Kids (⚽) та Adult (🍺)
- **3 рівні** з перешкодами в кожному
- **Відчеплення від маркера** — гра продовжується після фіксації
- **Промокоди** при голі
- **PWA** — можна встановити на домашній екран

## 🚀 Швидкий старт

```bash
npm install
npm run dev
